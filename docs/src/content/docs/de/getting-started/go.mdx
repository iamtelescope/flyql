## Installation

```bash
go get github.com/iamtelescope/flyql/golang
```

## Eine Abfrage parsen

```go
package main

import (
	"fmt"
	flyql "github.com/iamtelescope/flyql/golang"
)

func main() {
	result, err := flyql.Parse("status = 200 and active")
	if err != nil {
		panic(err)
	}
	fmt.Printf("Parsed: %+v\n", result.Root)
}
```

`Parse` gibt ein `*ParseResult` mit einem `Root`-Knoten zurück, der den AST repräsentiert.

## SQL generieren

Verwende einen Generator, um den AST in eine parametrisierte WHERE-Klausel umzuwandeln. Jeder Datenbankdialekt hat seinen eigenen Generator mit dialektspezifischen Spaltentypen.

### ClickHouse

```go
package main

import (
	"fmt"
	flyql "github.com/iamtelescope/flyql/golang"
	"github.com/iamtelescope/flyql/golang/generators/clickhouse"
)

func main() {
	result, err := flyql.Parse("status >= 400 and host = 'prod*'")
	if err != nil {
		panic(err)
	}

	columns := map[string]*clickhouse.Column{
		"status": clickhouse.NewColumn(clickhouse.ColumnDef{Name: "status", Type: "UInt32"}),
		"host":   clickhouse.NewColumn(clickhouse.ColumnDef{Name: "host", Type: "String"}),
	}

	sql, err := clickhouse.ToSQLWhere(result.Root, columns)
	if err != nil {
		panic(err)
	}
	fmt.Println(sql)
}
```

Die Generatoren für PostgreSQL und StarRocks folgen demselben Muster – importiere stattdessen `generators/postgresql` oder `generators/starrocks`.

## Im Speicher matchen

Werte eine Abfrage gegen einen Datensatz aus, ohne SQL zu generieren:

```go
package main

import (
	"fmt"
	"github.com/iamtelescope/flyql/golang/matcher"
)

func main() {
	data := map[string]any{
		"status": 200,
		"active": true,
		"host":   "prod-api-01",
	}

	matches, err := matcher.Match("status = 200 and active", data)
	if err != nil {
		panic(err)
	}
	fmt.Printf("Matches: %v\n", matches) // true
}
```

## Spaltenausdrücke parsen

Parse einen Spaltenausdruck in strukturierte Daten mit Segmenten, Transformern und Aliasen.
Das zweite Argument ist eine `Capabilities`-Struktur, die steuert, welche Funktionen aktiviert sind.
Transformers sind standardmäßig deaktiviert.

```go
package main

import (
	"fmt"
	"github.com/iamtelescope/flyql/golang/columns"
)

func main() {
	// Einfache Spalten parsen (Transformers sind standardmäßig deaktiviert)
	parsed, err := columns.Parse("message, status", columns.Capabilities{})
	if err != nil {
		panic(err)
	}
	for _, col := range parsed {
		fmt.Printf("%s (display: %q, segments: %v)\n", col.Name, col.DisplayName, col.Segments)
	}

	// Transformers über Capabilities aktivieren
	caps := columns.Capabilities{Transformers: true}
	withTransforms, err := columns.Parse("message|chars(25) as msg, status", caps)
	if err != nil {
		panic(err)
	}
	fmt.Println(withTransforms[0].Transformers)

	// Oder direkt als JSON für API-Antworten serialisieren
	jsonBytes, _ := columns.ParseToJSON("message, status|upper", caps)
	fmt.Println(string(jsonBytes))
}
```

## Parametrisierte Abfragen

Verwende `$name`- und `$1`-Platzhalter und löse sie zur Laufzeit mit `BindParams()` auf:

```go
package main

import (
	"fmt"
	flyql "github.com/iamtelescope/flyql/golang"
	"github.com/iamtelescope/flyql/golang/generators/clickhouse"
)

func main() {
	// Eine Abfrage mit Parameter-Platzhaltern parsen
	result, err := flyql.Parse(`status = $code and env in [$env, 'staging']`)
	if err != nil {
		panic(err)
	}

	// Konkrete Werte an die Parameter binden
	if err := flyql.BindParams(result.Root, map[string]any{
		"code": 200,
		"env":  "prod",
	}); err != nil {
		panic(err)
	}

	// SQL generieren
	columns := map[string]*clickhouse.Column{
		"status": clickhouse.NewColumn(clickhouse.ColumnDef{Name: "status", Type: "Int32"}),
		"env":    clickhouse.NewColumn(clickhouse.ColumnDef{Name: "env", Type: "String"}),
	}

	sql, err := clickhouse.ToSQLWhere(result.Root, columns)
	if err != nil {
		panic(err)
	}
	fmt.Println(sql)
	// (status = 200 AND env IN ('prod', 'staging'))
}
```

## Transformers in Abfragen

Parse eine Abfrage mit Transformern und generiere SQL:

```go
package main

import (
	"fmt"
	flyql "github.com/iamtelescope/flyql/golang"
	clickhousegen "github.com/iamtelescope/flyql/golang/generators/clickhouse"
)

func main() {
	result, _ := flyql.Parse("message|upper = 'ERROR'")
	columns := map[string]*clickhousegen.Column{
		"message": clickhousegen.NewColumn("message", false, "String", nil),
	}
	sql, _ := clickhousegen.ToSQLWhere(result.Root, columns)
	fmt.Println(sql) // equals(upper(message), 'ERROR')
}
```
