### Operadores JSONB

- `->` devuelve JSONB y se usa para el recorrido intermedio de la ruta
- `->>` devuelve texto y se usa para extraer el valor final
- `jsonb_typeof()` comprueba el tipo JSON antes de comparar

### Configuración de columnas

Las columnas de PostgreSQL no tienen parámetro `jsonString`; el tipo JSONB se detecta a partir del tipo de columna:

```go
// Go
columns := map[string]*postgresql.Column{
    "message": postgresql.NewColumn(postgresql.ColumnDef{Name: "message", Type: "varchar"}),
    "data":    postgresql.NewColumn(postgresql.ColumnDef{Name: "data", Type: "jsonb"}),
}
```

```javascript
// JavaScript
const columns = {
    message: newColumn({ name: "message", type: "varchar" }),
    data:    newColumn({ name: "data", type: "jsonb" }),
}
```
