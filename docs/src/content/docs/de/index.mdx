FlyQL ist eine leichtgewichtige Abfragesprache, die in injektionssicheres SQL für ClickHouse, PostgreSQL und StarRocks transpiliert wird, mit Implementierungen in Go, Python und JavaScript.

## Was ist FlyQL?

FlyQL ist eine einfache, gut lesbare Abfragesprache zum Filtern von Daten. Schreibe eine Abfrage wie `status >= 400 and host = prod*`, und FlyQL transpiliert sie in parametriertes SQL für deine Datenbank oder wertet sie im Speicher gegen Datensätze aus.

<FlyqlCode code={landingIntroQuery} />

Diese Abfrage funktioniert identisch, egal ob dein Backend ClickHouse, PostgreSQL, StarRocks oder einen Datensatz im Speicher verwendet.

## Hauptmerkmale

- **Injektionssicher per Design**: Die Grammatik kann keine Subqueries, `JOIN`s, `UNION`s oder DDL ausdrücken. Sicherheit ist strukturell verankert, nicht nur defensiv umgesetzt. Das gesamte generierte SQL verwendet parametrisierte Werte.
- **Mehrere SQL-Dialekte**: Eine Abfragesprache, dialektspezifische Ausgabe. FlyQL generiert korrektes SQL für ClickHouse, PostgreSQL und StarRocks mit passender Typbehandlung für JSON-Pfade, Map-Zugriffe und Arrays.
- **Mehrere Programmiersprachen**: Vollständige Implementierungen in Go, Python und JavaScript. Parse im Frontend, validiere im Backend, generiere SQL überall, gleiche Abfrage, gleiches Ergebnis.
- **Keine Abhängigkeiten**: Ein handgeschriebener State-Machine-Parser ohne externe Abhängigkeiten in jeder Sprache. Leicht integrierbar und konfliktfrei.
