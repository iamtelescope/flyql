FlyQL 是一种轻量级查询语言，可转换为适用于 ClickHouse、PostgreSQL 和 StarRocks 的防注入 SQL，并提供 Go、Python 和 JavaScript 实现。

## 什么是 FlyQL？

FlyQL 是一种简单、易读的查询语言，专门用于数据过滤。你可以写出像 `status >= 400 and host = prod*` 这样的查询，然后让 FlyQL 将其转换为数据库可用的 SQL，或者直接在内存中匹配数据记录。

<FlyqlCode code={landingIntroQuery} />

无论你的后端使用 ClickHouse、PostgreSQL、StarRocks，还是内存中的数据集，这条查询的行为都保持一致。

## 核心特性

- **天然防注入**：语法本身无法表达子查询、JOIN、UNION 或 DDL。安全性来自语言设计，而不是额外防御。所有生成的 SQL 都会安全处理值。
- **多 SQL 方言支持**：同一套查询语法，可输出不同数据库的正确 SQL。FlyQL 能针对 ClickHouse、PostgreSQL 和 StarRocks 生成符合各自特性的语句，包括 JSON 路径、Map 访问和数组处理。
- **多语言实现**：提供 Go、Python 和 JavaScript 的完整实现。你可以在前端解析、在后端校验、在任意位置生成 SQL，同一条查询在不同环境中保持一致。
- **零依赖**：所有实现都使用手写状态机解析器，不依赖外部库。集成简单，不容易引入额外冲突。
