description: Instala FlyQL y empieza a filtrar datos en Go, Python o JavaScript

FlyQL está disponible en tres lenguajes. Elige el que encaje con tu stack:

| Lenguaje | Instalación | Parser | Generadores | Evaluación en memoria |
|----------|-------------|--------|-------------|-----------------------|
| [Go](/es/getting-started/go/) | `go get` | Sí | ClickHouse, PostgreSQL, StarRocks | Sí |
| [Python](/es/getting-started/python/) | `pip install` (git) | Sí | ClickHouse, PostgreSQL, StarRocks | Sí |
| [JavaScript](/es/getting-started/javascript/) | `npm install` | Sí | ClickHouse, PostgreSQL, StarRocks | Sí |

Cada guía cubre la instalación, el parseo de una consulta, la generación de SQL y, cuando está disponible, la evaluación en memoria.
