### JSONB ऑपरेटर

- `->` JSONB लौटाता है (मध्यवर्ती पथ ट्रैवर्सल के लिए उपयोग)
- `->>` टेक्स्ट लौटाता है (अंतिम मान निष्कर्षण के लिए उपयोग)
- `jsonb_typeof()` तुलना से पहले JSON प्रकार की जाँच करता है

### कॉलम सेटअप

PostgreSQL कॉलम में `jsonString` पैरामीटर नहीं होता — JSONB प्रकार कॉलम प्रकार से पता लगाया जाता है:

```go
// Go
columns := map[string]*postgresql.Column{
    "message": postgresql.NewColumn(postgresql.ColumnDef{Name: "message", Type: "varchar"}),
    "data":    postgresql.NewColumn(postgresql.ColumnDef{Name: "data", Type: "jsonb"}),
}
```

```javascript
// JavaScript
const columns = {
    message: newColumn({ name: "message", type: "varchar" }),
    data:    newColumn({ name: "data", type: "jsonb" }),
}
```
