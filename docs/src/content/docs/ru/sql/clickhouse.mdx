ClickHouse использует имена колонок без кавычек, `match()` для регулярных выражений и `LIKE` для поиска по префиксу.

## Базовые примеры

| FlyQL | Генерируемый SQL |
|-------|------------------|
| `message = 'hello'` | `message = 'hello'` |
| `message != 'hello'` | `message != 'hello'` |
| `count > 10` | `count > 10` |
| `count <= 100` | `count <= 100` |
| `message ~ 'error.*'` | `match(message, 'error.*')` |
| `message !~ 'test.*'` | `not match(message, 'test.*')` |
| `host = prod*` | `host LIKE 'prod%'` |
