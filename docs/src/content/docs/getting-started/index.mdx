title: Getting Started
description: Install FlyQL and start filtering data in Go, Python, or JavaScript
---

FlyQL is available in three languages. Pick the one that matches your stack:

| Language | Install | Parser | Generators | Matcher |
|----------|---------|--------|-----------|---------|
| [Go](/getting-started/go/) | `go get` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [Python](/getting-started/python/) | `pip install` (git) | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [JavaScript](/getting-started/javascript/) | `npm install` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |

Each guide covers installation, parsing a query, generating SQL, and (where available) matching in-memory data.
