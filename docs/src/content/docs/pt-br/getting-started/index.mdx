description: Instale o FlyQL e comece a filtrar dados em Go, Python ou JavaScript
---

O FlyQL está disponível em três linguagens. Escolha a que combina com seu stack:

| Linguagem | Instalação | Parser | Geradores | Matcher |
|----------|---------|--------|-----------|---------|
| [Go](/pt-br/getting-started/go/) | `go get` | Sim | ClickHouse, PostgreSQL, StarRocks | Sim |
| [Python](/pt-br/getting-started/python/) | `pip install` | Sim | ClickHouse, PostgreSQL, StarRocks | Sim |
| [JavaScript](/pt-br/getting-started/javascript/) | `npm install` | Sim | ClickHouse, PostgreSQL, StarRocks | Sim |

Cada guia cobre instalação, parse de uma consulta, geração de SQL e (quando disponível) matching de dados em memória.
