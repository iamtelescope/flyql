---

Редактор FlyQL использует CSS-переменные для всех визуальных стилей. Переопределяйте любую переменную `--flyql-*`, чтобы подстроить внешний вид под дизайн вашего приложения.

## CSS-переменные

### Цвета

| Переменная | Светлая тема | Тёмная тема | Описание |
|------------|--------------|-------------|----------|
| `--flyql-bg` | `#ffffff` | `#1e1e1e` | Фон редактора |
| `--flyql-text` | `#1e1e1e` | `#d4d4d4` | Цвет текста |
| `--flyql-border` | `#d4d4d4` | `#525252` | Цвет рамки |
| `--flyql-border-focus` | `#075985` | `#0369a1` | Рамка при фокусе |
| `--flyql-placeholder-color` | `#a0a0a0` | `#676767` | Текст-подсказка |
| `--flyql-error-color` | `#ff0000` | `#f48771` | Подсветка ошибки |

### Подсветка синтаксиса

| Переменная | Светлая тема | Тёмная тема | Описание |
|------------|--------------|-------------|----------|
| `--flyql-key-color` | `#0451a5` | `#6e9fff` | Имена столбцов/ключей |
| `--flyql-operator-color` | `#0089ab` | `#0089ab` | Операторы (`=`, `!=`, `and` и т.д.) |
| `--flyql-value-color` | `#8b0000` | `#ce9178` | Строковые значения |
| `--flyql-number-color` | `#098658` | `#b5cea8` | Числовые значения |

### Панель подсказок

| Переменная | Светлая тема | Тёмная тема | Описание |
|------------|--------------|-------------|----------|
| `--flyql-dropdown-bg` | `#ffffff` | `#252526` | Фон панели |
| `--flyql-dropdown-border` | `#d4d4d4` | `#454545` | Рамка панели |
| `--flyql-dropdown-item-hover` | `#f7f7f7` | `#2e3132` | Элемент при наведении |
| `--flyql-dropdown-item-active` | `#eef4fb` | `#1a2a3a` | Выбранный элемент |

### Типографика

| Переменная | По умолчанию | Описание |
|------------|--------------|----------|
| `--flyql-font-family` | `'Open Sans', sans-serif` | Шрифт интерфейса |
| `--flyql-code-font-family` | `monospace` | Шрифт кода/запроса |
| `--flyql-font-size` | `13px` | Базовый размер шрифта |

## Светлая тема

Светлая тема используется по умолчанию. Дополнительные классы не нужны:

```html
<FlyqlEditor v-model="query" :columns="columns" />
```

## Тёмная тема

Используйте проп `dark` для включения встроенной тёмной темы:

```html
<FlyqlEditor v-model="query" :columns="columns" :dark="isDark" />
<FlyqlColumns v-model="expr" :columns="columns" :dark="isDark" />
```

Проп `dark` применяет класс `flyql-dark` к панели подсказок, которая телепортируется в `<body>` и не может наследовать стили от родительских обёрток. Само поле ввода редактора продолжает наследовать CSS-переменные от своих предков как обычно, поэтому переопределения на уровне обёртки по-прежнему работают.

Альтернативно можно добавить класс `flyql-dark` к `<html>` или `<body>`, что позволит стилям достичь и редактора, и телепортированной панели через наследование CSS-переменных:

```javascript
document.documentElement.classList.toggle('flyql-dark', isDark)
```

## Пользовательская тема

Переопределите нужные переменные в вашем CSS:

```css
/* Пользовательская тема */
:root {
    --flyql-border-focus: #7c3aed;
    --flyql-key-color: #7c3aed;
    --flyql-operator-color: #059669;
    --flyql-dropdown-item-active: #ede9fe;
}

/* Пользовательские тёмные стили */
.flyql-dark {
    --flyql-bg: #0f172a;
    --flyql-text: #e2e8f0;
    --flyql-border: #334155;
    --flyql-key-color: #a78bfa;
    --flyql-dropdown-bg: #1e293b;
}
```

## Локальная тема

Оформите конкретный экземпляр редактора, обернув его в контейнер с переопределениями:

```html
<div class="my-editor-theme">
    <FlyqlEditor v-model="query" :columns="columns" />
</div>

<style>
.my-editor-theme {
    --flyql-font-size: 15px;
    --flyql-border-focus: #2563eb;
    --flyql-bg: #f8fafc;
}
</style>
```

CSS-переменные каскадируются, поэтому переопределения на любом предке применяются ко всем компонентам FlyQL внутри него.
