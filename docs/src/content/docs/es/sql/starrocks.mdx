## Tipos complejos anidados

El generador de StarRocks trata una ruta segmentada como perteneciente a un único tipo contenedor: actualmente no desciende por anidamientos mixtos como `Map(K, Struct)`, `Array(Map)` o campos `Struct` que a su vez sean `Map`/`Array`. Para estas columnas, aplana la ruta o expone el valor interno como una columna aparte.

### Configuración de columnas

{/* TODO: native-speaker review */}
Las columnas text/varchar con contenido JSON se declaran con el tipo bruto sintético `jsonstring` (igual que en ClickHouse):

```go
// Go
columns := map[string]*starrocks.Column{
    "message": starrocks.NewColumn(starrocks.ColumnDef{Name: "message", Type: "VARCHAR(255)"}),
    "data":    starrocks.NewColumn(starrocks.ColumnDef{Name: "data", Type: "JSON"}),      // native JSON
    "log":     starrocks.NewColumn(starrocks.ColumnDef{Name: "log", Type: "jsonstring"}),       // JSON string
}
```

```javascript
// JavaScript
const columns = {
    message: newColumn({ name: "message", type: "VARCHAR(255)" }),
    data:    newColumn({ name: "data", type: "JSON" }),      // native JSON
    log:     newColumn({ name: "log", type: "jsonstring" }),       // JSON string
}
```
