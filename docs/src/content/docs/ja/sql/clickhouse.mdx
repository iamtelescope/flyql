## JSON カラム

ClickHouse は 2 種類の JSON カラム型をサポートし、それぞれ異なる SQL を生成します。

### ネイティブ JSON 型

`JSON` 型として宣言されたカラムは、バッククォート付きセグメントのドット記法を使います。

<SqlExamplesTable rows={jsonNative} />

### JSON String 型

JSON を文字列として保持するカラム（例: JSON 内容を持つ `String` 型）は、型チェックのために `multiIf` でラップした `JSONExtract` 関数を使います。

<SqlExamplesTable rows={jsonString} />

## Map カラム

Map カラムでは、角括弧記法と `equals()` / `notEquals()` 関数を使います。

<SqlExamplesTable rows={mapColumns} />

## Array カラム

FlyQL の配列インデックスは 0 始まりです。ClickHouse 配列は SQL レイヤーで 1 始まりであるため、ClickHouse ジェネレーターは SQL を生成する際にインデックスに 1 を加算します。

<SqlExamplesTable rows={arrayColumns} />

### カラム設定

{/* TODO: native-speaker review */}
JSON 内容を持つ text/varchar カラムは、合成的な生型 `jsonstring` として宣言します。

```go
// Go
columns := map[string]*clickhouse.Column{
    "message": clickhouse.NewColumn(clickhouse.ColumnDef{Name: "message", Type: "String"}),
    "data":    clickhouse.NewColumn(clickhouse.ColumnDef{Name: "data", Type: "JSON"}),      // native JSON
    "log":     clickhouse.NewColumn(clickhouse.ColumnDef{Name: "log", Type: "jsonstring"}),       // JSON string
}
```

```javascript
// JavaScript
const columns = {
    message: newColumn({ name: "message", type: "String" }),
    data:    newColumn({ name: "data", type: "JSON" }),      // native JSON
    log:     newColumn({ name: "log", type: "jsonstring" }),       // JSON string
}
```
