## JSON-столбцы

StarRocks поддерживает два типа JSON-столбцов с разным генерируемым SQL.

### Нативный тип JSON

Нативные JSON-столбцы используют оператор `->` с JSON-ключами в строковом виде:

<SqlExamplesTable rows={jsonNative} />

### JSON в виде строки

<SqlExamplesTable rows={jsonString} />

## Вложенные сложные типы

Генератор StarRocks трактует сегментированный путь как принадлежащий одному контейнерному типу — он пока не спускается через смешанное вложение вроде `Map(K, Struct)`, `Array(Map)` или `Struct`-поля, которые сами являются `Map`/`Array`. Для таких колонок выравнивайте путь или выносите вложенное значение в отдельную колонку.

### Настройка столбцов

Колонки text/varchar с содержимым JSON объявляются с синтетическим сырым типом `jsonstring` (аналогично ClickHouse):

```go
// Go
columns := map[string]*starrocks.Column{
    "message": starrocks.NewColumn(starrocks.ColumnDef{Name: "message", Type: "VARCHAR(255)"}),
    "data":    starrocks.NewColumn(starrocks.ColumnDef{Name: "data", Type: "JSON"}),      // native JSON
    "log":     starrocks.NewColumn(starrocks.ColumnDef{Name: "log", Type: "jsonstring"}),       // JSON string
}
```

```javascript
// JavaScript
const columns = {
    message: newColumn({ name: "message", type: "VARCHAR(255)" }),
    data:    newColumn({ name: "data", type: "JSON" }),      // native JSON
    log:     newColumn({ name: "log", type: "jsonstring" }),       // JSON string
}
```
