## इंस्टॉलेशन

Python के लिए FlyQL PyPI पर उपलब्ध है:

```bash
pip install flyql
```

यह अपनी डिपेंडेंसी `google-re2` के साथ `flyql` पैकेज इंस्टॉल करता है।

**आवश्यकताएँ:** Python 3.10+

## क्वेरी पार्स करें

```python
from flyql import parse

result = parse("status = 200 and active")
print(result.root)
```

`parse()` AST का प्रतिनिधित्व करने वाले `root` नोड के साथ एक `Parser` ऑब्जेक्ट लौटाता है। अमान्य इनपुट पर यह `ParserError` उत्पन्न करता है।

## SQL उत्पन्न करें

AST को पैरामीटराइज़्ड WHERE क्लॉज़ में बदलने के लिए जनरेटर का उपयोग करें। प्रत्येक डेटाबेस बोली का अपना जनरेटर है।

### ClickHouse

```python
from flyql import parse
from flyql.generators.clickhouse.generator import to_sql_where
from flyql.generators.clickhouse.column import Column

result = parse("status >= 400 and host like 'prod%'")

columns = {
    "status": Column("status", False, "UInt32"),
    "host": Column("host", False, "String"),
}

sql = to_sql_where(result.root, columns)
print(sql)
```

PostgreSQL और StarRocks जनरेटर समान पैटर्न का पालन करते हैं — इसके बजाय `flyql.generators.postgresql` या `flyql.generators.starrocks` से इम्पोर्ट करें।

## इन-मेमोरी मैच

SQL उत्पन्न किए बिना डेटा रिकॉर्ड के विरुद्ध क्वेरी का मूल्यांकन करें:

```python
from flyql import parse
from flyql.matcher import Evaluator, Record


result = parse("status = 200 and active")

data = {
    "status": 200,
    "active": True,
    "host": "prod-api-01",
}

evaluator = Evaluator()
matches = evaluator.evaluate(result.root, Record(data))
print(f"Matches: {matches}")  # True
```

## कॉलम एक्सप्रेशन पार्स करें

कॉलम एक्सप्रेशन को सेगमेंट, ट्रांसफ़ॉर्मर, और उपनामों के साथ संरचित डेटा में पार्स करें:

```python
from flyql.columns import parse, parse_to_json

# Parse basic columns (transformers disabled by default)
parsed = parse("message, status")
for col in parsed:
    print(f"{col.name} (display: {col.display_name!r}, segments: {col.segments})")

# Enable transformers via capabilities
with_transforms = parse("message|chars(25) as msg, status", capabilities={"transformers": True})

# Or serialize directly to JSON for API responses
json_str = parse_to_json("message, status|upper", capabilities={"transformers": True})
print(json_str)
```

## पैरामीटराइज़्ड क्वेरी

`$name` और `$1` प्लेसहोल्डर का उपयोग करें, फिर रनटाइम पर `bind_params()` के साथ उन्हें हल करें:

```python
from flyql import parse, bind_params
from flyql.generators.clickhouse.generator import to_sql_where
from flyql.generators.clickhouse.column import Column

# Named parameters
ast = parse("status = $code and env = $env")
bind_params(ast.root, {"code": 200, "env": "prod"})

columns = {
    "status": Column("status", False, "UInt32"),
    "env": Column("env", False, "String"),
}
sql = to_sql_where(ast.root, columns)
print(sql)

# Positional parameters
ast = parse("status = $1 and env = $2")
bind_params(ast.root, {"1": 200, "2": "prod"})
```

पूर्ण संदर्भ के लिए [पैरामीटर](/hi/syntax/parameters/) देखें।

## क्वेरी में ट्रांसफ़ॉर्मर

ट्रांसफ़ॉर्मर के साथ क्वेरी पार्स करें और SQL उत्पन्न करें:

```python
from flyql import parse
from flyql.generators.clickhouse.generator import to_sql_where, Column

# Parse a transformer query
result = parse("message|upper = 'ERROR'")

# Generate SQL for ClickHouse
columns = {"message": Column("message", False, "String")}
sql = to_sql_where(result.root, columns)
print(sql)  # equals(upper(message), 'ERROR')
```

पूर्ण सिंटैक्स संदर्भ के लिए [ट्रांसफ़ॉर्मर](/hi/syntax/transformers/) देखें।
