### Operadores JSONB

- `->` retorna JSONB (usado para travessia intermediária de caminho)
- `->>` retorna texto (usado para extração final do valor)
- `jsonb_typeof()` verifica o tipo JSON antes da comparação

### Configuração de Colunas

Colunas do PostgreSQL não possuem o parâmetro `jsonString` — o tipo JSONB é detectado a partir do tipo da coluna:

```go
// Go
columns := map[string]*postgresql.Column{
    "message": postgresql.NewColumn(postgresql.ColumnDef{Name: "message", Type: "varchar"}),
    "data":    postgresql.NewColumn(postgresql.ColumnDef{Name: "data", Type: "jsonb"}),
}
```

```javascript
// JavaScript
const columns = {
    message: newColumn({ name: "message", type: "varchar" }),
    data:    newColumn({ name: "data", type: "jsonb" }),
}
```
