title: शुरू करें
description: FlyQL इंस्टॉल करें और Go, Python, या JavaScript में डेटा फ़िल्टर करना शुरू करें
---

FlyQL तीन भाषाओं में उपलब्ध है। अपने स्टैक से मेल खाने वाली चुनें:

| भाषा | इंस्टॉल | पार्सर | जनरेटर | मैचर |
|----------|---------|--------|-----------|---------|
| [Go](/hi/getting-started/go/) | `go get` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [Python](/hi/getting-started/python/) | `pip install` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [JavaScript](/hi/getting-started/javascript/) | `npm install` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |

प्रत्येक गाइड इंस्टॉलेशन, क्वेरी पार्सिंग, SQL जनरेशन, और (जहाँ उपलब्ध हो) इन-मेमोरी डेटा मैचिंग को कवर करता है।
