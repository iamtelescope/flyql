FlyQL — лёгкий язык запросов, который преобразуется в SQL с защитой от инъекций для ClickHouse, PostgreSQL и StarRocks, с реализациями на Go, Python и JavaScript.

## Что такое FlyQL?

FlyQL — это простой и удобочитаемый язык запросов, предназначенный для фильтрации данных. Вы можете написать запрос вроде `status>=400 and host=prod*`, а FlyQL преобразует его в SQL для вашей базы данных или выполнит сопоставление прямо в памяти.

```
status>=400 and host=prod* and not debug
```

Этот запрос работает одинаково независимо от того, используете ли вы ClickHouse, PostgreSQL, StarRocks или набор данных в памяти.

## Ключевые возможности

- **Защита от инъекций на уровне дизайна**: грамматика не позволяет выражать подзапросы, `JOIN`, `UNION` или DDL. Безопасность обеспечивается структурой языка, а не дополнительными защитными мерами.
- **Поддержка нескольких диалектов SQL**: один язык запросов, но разный корректный SQL для каждой СУБД. FlyQL генерирует SQL для ClickHouse, PostgreSQL и StarRocks с учётом особенностей JSON, Map и массивов.
- **Реализации на нескольких языках**: доступны полноценные реализации на Go, Python и JavaScript. Можно парсить на фронтенде, валидировать на бэкенде и генерировать SQL где угодно.
- **Без зависимостей**: во всех реализациях используется вручную написанный парсер на конечном автомате без внешних библиотек.

## Быстрый пример

```
service!=api and (status>=400 or level=error) and message~"timeout.*"
```

| Компонент | Значение |
|-----------|----------|
| `service!=api` | поле `service` не равно `"api"` |
| `status>=400` | `status` больше или равен `400` |
| `level=error` | поле `level` равно `"error"` |
| `message~"timeout.*"` | `message` соответствует регулярному выражению |
| `and`, `or` | булева логика с приоритетом |
| `( )` | явная группировка |

## Поддерживаемые языки

| Язык | Парсер | SQL-генераторы | Сопоставление в памяти |
|------|--------|----------------|------------------------|
| Go | Да | ClickHouse, PostgreSQL, StarRocks | Да |
| Python | Да | ClickHouse, PostgreSQL, StarRocks | Да |
| JavaScript | Да | ClickHouse, PostgreSQL, StarRocks | Да |
