PostgreSQL использует двойные кавычки для идентификаторов, `~` для регулярных выражений, `AND`/`OR` в верхнем регистре и операторы JSONB для вложенных полей.

## Базовые примеры

| FlyQL | Генерируемый SQL |
|-------|------------------|
| `message='hello'` | `"message" = 'hello'` |
| `message!='hello'` | `"message" != 'hello'` |
| `count>10` | `"count" > 10` |
| `count<=100` | `"count" <= 100` |
| `message~'error.*'` | `"message" ~ 'error.*'` |
| `message!~'test.*'` | `"message" !~ 'test.*'` |
| `host=prod*` | `"host" LIKE 'prod%'` |
