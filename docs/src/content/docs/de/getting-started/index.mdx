title: Erste Schritte
description: Installiere FlyQL und beginne mit dem Filtern von Daten in Go, Python oder JavaScript
---

FlyQL ist in drei Sprachen verfügbar. Wähle die Variante, die zu deinem Stack passt:

| Sprache | Installation | Parser | Generatoren | Matcher |
|----------|-------------|--------|-------------|---------|
| [Go](/de/getting-started/go/) | `go get` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [Python](/de/getting-started/python/) | `pip install` (git) | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [JavaScript](/de/getting-started/javascript/) | `npm install` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |

Jede Anleitung behandelt Installation, das Parsen einer Abfrage, die SQL-Generierung und – wo verfügbar – das Matching von In-Memory-Daten.
