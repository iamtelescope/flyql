---
title: StarRocks
description: Примеры генерируемого SQL для StarRocks
---

StarRocks использует идентификаторы в обратных кавычках, `regexp()` для регулярных выражений и `->` с JSON-ключами в строковом виде для доступа к JSON.

## Базовые примеры

| FlyQL | Генерируемый SQL |
|-------|------------------|
| `message='hello'` | `` `message` = 'hello' `` |
| `message!='hello'` | `` `message` != 'hello' `` |
| `count>10` | `` `count` > 10 `` |
| `count<=100` | `` `count` <= 100 `` |
| `message~'error.*'` | `` regexp(`message`, 'error.*') `` |
| `message!~'test.*'` | `` not regexp(`message`, 'test.*') `` |
| `host=prod*` | `` `host` LIKE 'prod%' `` |
