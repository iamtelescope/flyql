As consultas FlyQL consistem em uma ou mais condições conectadas por <FlyqlToken type="operator">and</FlyqlToken> / <FlyqlToken type="operator">or</FlyqlToken>, opcionalmente negadas com <FlyqlToken type="operator">not</FlyqlToken>. Cada condição é uma comparação ou uma verificação de valor truthy.

## Estrutura Básica da Consulta

<FlyqlCode code={`status = 200 and active and not archived`} />

- <FlyqlCode code={`status = 200`} inline /> — o campo <FlyqlToken type="key">status</FlyqlToken> é igual a <FlyqlToken type="value">200</FlyqlToken>
- <FlyqlCode code={`active`} inline /> — o campo <FlyqlToken type="key">active</FlyqlToken> possui um valor truthy
- <FlyqlCode code={`not archived`} inline /> — o campo <FlyqlToken type="key">archived</FlyqlToken> é falsy (null, vazio, zero ou false)

## Mais Exemplos

<FlyqlCode code={`service != 'api' or user = "john doe"`} />

<FlyqlCode code={`message ~ "error.*" and not debug`} />

<FlyqlCode code={`(a = 1 or b = 2) and not (c = 3 and d = 4)`} />

<FlyqlCode code={`status in [200, 201] and method not in ['DELETE', 'PUT']`} />

<FlyqlCode code={`message has 'error' and tags not has 'debug'`} />

## Regras Gerais

- **Chaves isoladas** — Uma chave sem operador é tratada como verificação truthy: <FlyqlCode code={`active`} inline />
- **Comparações** — Uma chave com operador deve ter um valor correspondente: <FlyqlCode code={`status = 200`} inline />
- **Espaços em branco** — Espaços ao redor dos operadores são opcionais: <FlyqlCode code={`status=200`} inline /> e <FlyqlCode code={`status = 200`} inline /> são equivalentes
- **Avaliação da esquerda para a direita** — não há precedência entre <FlyqlToken type="operator">and</FlyqlToken> e <FlyqlToken type="operator">or</FlyqlToken>: <FlyqlCode code={`a = 1 or b = 2 and c = 3`} inline /> é interpretado como <FlyqlCode code={`(a = 1 or b = 2) and c = 3`} inline />, e **não** como `a = 1 or (b = 2 and c = 3)`
- **Parênteses** — Use <FlyqlToken type="operator">( )</FlyqlToken> sempre que o agrupamento for relevante: <FlyqlCode code={`a = 1 or (b = 2 and c = 3)`} inline />
