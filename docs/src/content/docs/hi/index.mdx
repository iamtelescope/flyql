**FlyQL** प्रोजेक्ट प्रलेखन में आपका स्वागत है। वर्तमान संस्करण: **0.0.32** ([परिवर्तन लॉग](/hi/changelog))।

FlyQL डेटा फ़िल्टरिंग के लिए डिज़ाइन की गई एक हल्की, मानव-पठनीय क्वेरी भाषा है। यह **ClickHouse**, **PostgreSQL**, और **StarRocks** के लिए इंजेक्शन-प्रूफ़ पैरामीटराइज़्ड SQL में ट्रांसपाइल होती है — **Go**, **Python**, और **JavaScript** में पूर्ण कार्यान्वयन के साथ। एक बार क्वेरी लिखें, और FlyQL आपके डेटाबेस के लिए बोली-विशिष्ट SQL उत्पन्न करता है — या इसे इन-मेमोरी डेटा रिकॉर्ड के विरुद्ध मूल्यांकित करता है।

यह क्वेरी समान रूप से काम करती है चाहे आपका बैकएंड ClickHouse हो, PostgreSQL हो, StarRocks हो, या इन-मेमोरी डेटासेट हो।

## प्रमुख विशेषताएँ

- **डिज़ाइन द्वारा इंजेक्शन-प्रूफ़** — व्याकरण सबक्वेरी, JOIN, UNION, या DDL व्यक्त नहीं कर सकता। सुरक्षा संरचनात्मक है, रक्षात्मक नहीं। सभी उत्पन्न SQL पैरामीटराइज़्ड मानों का उपयोग करता है।
- **बहु-बोली SQL** — एक क्वेरी भाषा, बोली-विशिष्ट आउटपुट। उचित प्रकार हैंडलिंग (JSON पथ, मैप एक्सेस, ऐरे हैंडलिंग) के साथ ClickHouse, PostgreSQL, और StarRocks के लिए सही SQL उत्पन्न करता है।
- **बहु-भाषा** — Go, Python, और JavaScript में पूर्ण कार्यान्वयन। फ़्रंटएंड पर पार्स करें, बैकएंड पर मान्य करें, कहीं भी SQL उत्पन्न करें — समान क्वेरी, समान परिणाम।
- **शून्य डिपेंडेंसी** — किसी भी भाषा में बिना बाहरी डिपेंडेंसी के हाथ से लिखा गया स्टेट-मशीन पार्सर। एकीकृत करना सरल, कोई विरोध नहीं।
