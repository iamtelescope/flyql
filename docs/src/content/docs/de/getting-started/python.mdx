## Installation

FlyQL für Python wird aus git installiert (nicht auf PyPI veröffentlicht):

```bash
pip install "flyql @ git+https://github.com/iamtelescope/flyql.git#subdirectory=python"
```

Dadurch wird das Paket `flyql` zusammen mit seiner Abhängigkeit `google-re2` installiert.

**Anforderungen:** Python 3.10+

## Eine Abfrage parsen

```python
from flyql import parse

result = parse("status = 200 and active")
print(result.root)
```

`parse()` gibt ein `Parser`-Objekt mit einem `root`-Knoten zurück, der den AST repräsentiert. Bei ungültiger Eingabe wird `ParserError` ausgelöst.

## SQL generieren

Verwende einen Generator, um den AST in eine parametrisierte WHERE-Klausel umzuwandeln. Jeder Datenbankdialekt hat seinen eigenen Generator.

### ClickHouse

```python
from flyql import parse
from flyql.generators.clickhouse.generator import to_sql_where
from flyql.generators.clickhouse.column import Column

result = parse("status >= 400 and host = 'prod*'")

columns = {
    "status": Column("status", False, "UInt32"),
    "host": Column("host", False, "String"),
}

sql = to_sql_where(result.root, columns)
print(sql)
```

Die Generatoren für PostgreSQL und StarRocks folgen demselben Muster – importiere stattdessen aus `flyql.generators.postgresql` oder `flyql.generators.starrocks`.

## Im Speicher matchen

Werte eine Abfrage gegen einen Datensatz aus, ohne SQL zu generieren:

```python
from flyql import parse
from flyql.matcher import Evaluator, Record


result = parse("status = 200 and active")

data = {
    "status": 200,
    "active": True,
    "host": "prod-api-01",
}

evaluator = Evaluator()
matches = evaluator.evaluate(result.root, Record(data))
print(f"Matches: {matches}")  # True
```

## Spaltenausdrücke parsen

Parse einen Spaltenausdruck in strukturierte Daten mit Segmenten, Transformern und Aliasen:

```python
from flyql.columns import parse, parse_to_json

# Einfache Spalten parsen (Transformers sind standardmäßig deaktiviert)
parsed = parse("message, status")
for col in parsed:
    print(f"{col.name} (display: {col.display_name!r}, segments: {col.segments})")

# Transformers über Capabilities aktivieren
with_transforms = parse("message|chars(25) as msg, status", capabilities={"transformers": True})

# Oder direkt als JSON für API-Antworten serialisieren
json_str = parse_to_json("message, status|upper", capabilities={"transformers": True})
print(json_str)
```

## Transformers in Abfragen

Parse eine Abfrage mit Transformern und generiere SQL:

```python
from flyql import parse
from flyql.generators.clickhouse.generator import to_sql_where, Column

# Eine Abfrage mit Transformern parsen
result = parse("message|upper = 'ERROR'")

# SQL für ClickHouse generieren
columns = {"message": Column("message", False, "String")}
sql = to_sql_where(result.root, columns)
print(sql)  # equals(upper(message), 'ERROR')
```
