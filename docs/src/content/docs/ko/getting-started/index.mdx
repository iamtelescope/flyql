title: 시작하기
description: FlyQL을 설치하고 Go, Python, JavaScript에서 데이터 필터링을 시작합니다
---

FlyQL은 세 가지 언어로 제공됩니다. 사용 중인 스택에 맞는 것을 선택하세요.

| 언어 | 설치 | Parser | Generator | Matcher |
|------|------|--------|-----------|---------|
| [Go](/ko/getting-started/go/) | `go get` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [Python](/ko/getting-started/python/) | `pip install` (git) | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [JavaScript](/ko/getting-started/javascript/) | `npm install` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |

각 가이드는 설치, 쿼리 parse, SQL 생성, 그리고 가능한 경우 메모리 내 데이터 매칭까지 다룹니다.
