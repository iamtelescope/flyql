title: Demarrage rapide
description: Installer FlyQL et commencer a filtrer des donnees en Go, Python ou JavaScript
---

FlyQL est disponible dans trois langages. Choisissez celui qui correspond a votre stack :

| Langage | Installation | Parseur | Generateurs | Matcher |
|---------|-------------|---------|-------------|---------|
| [Go](/fr/getting-started/go/) | `go get` | Oui | ClickHouse, PostgreSQL, StarRocks | Oui |
| [Python](/fr/getting-started/python/) | `pip install` | Oui | ClickHouse, PostgreSQL, StarRocks | Oui |
| [JavaScript](/fr/getting-started/javascript/) | `npm install` | Oui | ClickHouse, PostgreSQL, StarRocks | Oui |

Chaque guide couvre l'installation, l'analyse d'une requete, la generation de SQL et (lorsque disponible) le matching de donnees en memoire.
