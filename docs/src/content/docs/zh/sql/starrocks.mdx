title: StarRocks
description: StarRocks 的 SQL 生成示例
---

StarRocks 生成的 SQL 使用反引号包裹标识符，正则使用 `regexp()`，JSON 访问依赖 `->` 与 JSON 字符串形式的键。

## 基础示例

| FlyQL | 生成 SQL |
|-------|----------|
| `message = 'hello'` | `` `message` = 'hello' `` |
| `message != 'hello'` | `` `message` != 'hello' `` |
| `count > 10` | `` `count` > 10 `` |
| `count <= 100` | `` `count` <= 100 `` |
| `message ~ 'error.*'` | `` regexp(`message`, 'error.*') `` |
| `message !~ 'test.*'` | `` not regexp(`message`, 'test.*') `` |
| `host = prod*` | `` `host` LIKE 'prod%' `` |

## 布尔操作符

| FlyQL | 生成 SQL |
|-------|----------|
| `a = '1' and b > 10` | `` (`a` = '1' and `b` > 10) `` |
| `a = '1' or b = '2'` | `` (`a` = '1' or `b` = '2') `` |
| `not a = '1'` | `` not (`a` = '1') `` |

## 列表成员

| FlyQL | 生成 SQL |
|-------|----------|
| `status in [200, 201]` | `` `status` IN (200, 201) `` |
| `env not in ['dev', 'test']` | `` `env` NOT IN ('dev', 'test') `` |
| `status in []` | `0` |
| `status not in []` | `1` |

## 包含检查

| 列类型 | FlyQL | 生成 SQL |
|--------|-------|----------|
| String | `message has 'error'` | `` INSTR(`message`, 'error') > 0 `` |
| Array | `tags has 'web'` | `` array_contains(`tags`, 'web') `` |
| Map | `metadata has 'key'` | `` array_contains(map_keys(`metadata`), 'key') `` |
| Native JSON | `data has 'key'` | `` json_exists(`data`, concat('$.', 'key')) `` |

## JSON 列

StarRocks 支持原生 JSON 与字符串形式 JSON 两种模式。

### 原生 JSON

| FlyQL | 生成 SQL |
|-------|----------|
| `data.name = 'test'` | `` `data`->'\"name\"' = 'test' `` |
| `data.user.name = 'john'` | `` `data`->'\"user\"'->'\"name\"' = 'john' `` |

### 字符串形式 JSON

| FlyQL | 生成 SQL |
|-------|----------|
| `data.name = 'test'` | `` parse_json(`data`)->'\"name\"' = 'test' `` |
| `data.user.name = 'john'` | `` parse_json(`data`)->'\"user\"'->'\"name\"' = 'john' `` |

## Map 列

| FlyQL | 生成 SQL |
|-------|----------|
| `metadata.key1 = 'value1'` | `` `metadata`['key1'] = 'value1' `` |
| `metadata.key1 != 'value1'` | `` `metadata`['key1'] != 'value1' `` |
| `metadata.nested.key = 'value'` | `` `metadata`['nested']['key'] = 'value' `` |
| `metadata.pattern ~ 'test.*'` | `` `metadata`['pattern'] regexp 'test.*' `` |

## 数组列

StarRocks 数组下标是 0-based：

| FlyQL | 生成 SQL |
|-------|----------|
| `tags.0 = 'first'` | `` `tags`[0] = 'first' `` |
| `tags.1 = 'second'` | `` `tags`[1] = 'second' `` |

## 列定义示例

与 ClickHouse 类似，`jsonString` 参数控制 JSON 模式：

```go
columns := map[string]*starrocks.Column{
    "message": starrocks.NewColumn("message", false, "VARCHAR(255)", nil),
    "data":    starrocks.NewColumn("data", false, "JSON", nil),
    "log":     starrocks.NewColumn("log", true, "STRING", nil),
}
```

```javascript
const columns = {
    message: newColumn("message", false, "VARCHAR(255)", null),
    data:    newColumn("data", false, "JSON", null),
    log:     newColumn("log", true, "STRING", null),
}
```

## 转换器

| FlyQL | 生成 SQL |
|-------|----------|
| `status\|upper = "ERROR"` | `` UPPER(`status`) = 'ERROR' `` |
| `status\|lower = "info"` | `` LOWER(`status`) = 'info' `` |
| `message\|len > 100` | `` LENGTH(`message`) > 100 `` |
| `message\|lower\|len > 100` | `` LENGTH(LOWER(`message`)) > 100 `` |
| `tags\|split(",") has "urgent"` | `` array_contains(SPLIT(`tags`, ','), 'urgent') `` |
