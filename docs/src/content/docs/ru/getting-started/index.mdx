title: Быстрый старт
description: Установите FlyQL и начните фильтровать данные в Go, Python или JavaScript
---

FlyQL доступен на трёх языках. Выберите вариант, который лучше соответствует вашему стеку:

| Язык | Установка | Парсер | Генераторы | Сопоставление |
|------|-----------|--------|------------|---------------|
| [Go](/ru/getting-started/go/) | `go get` | Да | ClickHouse, PostgreSQL, StarRocks | Да |
| [Python](/ru/getting-started/python/) | `pip install` | Да | ClickHouse, PostgreSQL, StarRocks | Да |
| [JavaScript](/ru/getting-started/javascript/) | `npm install` | Да | ClickHouse, PostgreSQL, StarRocks | Да |

В каждом руководстве показаны установка, разбор запроса, генерация SQL и, где доступно, сопоставление данных в памяти.
