## Regex 매칭

값을 정규식과 비교하려면 `~`를, 매칭을 부정하려면 `!~`를 사용합니다.

```
message ~ "error.*timeout"
path !~ "^/health"
```

| 연산자 | 의미 |
|--------|------|
| `~` | 값이 regex 패턴과 일치함 |
| `!~` | 값이 regex 패턴과 일치하지 않음 |

### Regex 패턴 문법

Regex 문법은 대상 백엔드에 따라 달라집니다.

- **ClickHouse** — `match()` 함수 사용 (RE2 문법)
- **PostgreSQL** — `~`, `!~` 연산자 사용 (POSIX regex)
- **StarRocks** — `regexp` 함수 사용
- **In-memory matcher** — 호스트 언어의 regex 엔진 사용 (Go `regexp`, Python `re2`, JS `RegExp`)

패턴에 공백이나 특수 문자가 포함되면 따옴표로 감싸세요.

```
message ~ "connection (refused|reset)"
host ~ "^prod-[0-9]+"
```

### 대소문자 구분 없는 매칭

대소문자를 구분하지 않는 regex 매칭을 하려면 패턴 시작 부분에 `(?i)` 플래그를 사용합니다.

```
message ~ "(?i)error"
level ~ "(?i)^(warn|error|fatal)$"
```

인라인 플래그 `(?i)`는 regex 기능이며, 지원 여부는 백엔드마다 다릅니다.

| 백엔드 | `(?i)` 지원 여부 | 비고 |
|--------|------------------|------|
| **ClickHouse** | Yes | RE2 문법 |
| **PostgreSQL** | Yes | POSIX regex |
| **StarRocks** | Yes | RE2 문법 |
| **Go In-Memory Matcher** | Yes | Go `regexp` (RE2) |
| **Python In-Memory Matcher** | Yes | `google-re2` |
| **JavaScript In-Memory Matcher** | No | ECMAScript `RegExp`는 `(?i)` 인라인 플래그를 지원하지 않음 |

JavaScript In-Memory Matcher에서는 대신 문자 클래스 방식을 사용해 대소문자 구분 없는 매칭을 구현할 수 있습니다.

```
message ~ "[eE][rR][rR][oO][rR]"
```

또는 SQL 백엔드만 대상으로 한다면 `(?i)`는 세 가지 방언 모두에서 안정적으로 동작합니다.

대소문자 구분 없는 매칭과 부정을 결합하면 대소문자에 상관없이 특정 패턴을 제외할 수 있습니다.

```
message ~ "(?i)error" and message !~ "(?i)warning"
```

## 와일드카드 매칭

따옴표로 감싸지 않은 값의 끝에 `*`를 붙이면 글롭 스타일 접두사 매칭을 할 수 있습니다.

```
host = prod*
service = api-*
```

이는 주어진 접두사로 시작하는 모든 값과 일치합니다. 와일드카드는 값의 끝에서만 지원되며, `*prod` 또는 `pr*od`는 지원되지 않습니다.

### 와일드카드 vs. Regex

| 패턴 | 문법 | 예제 |
|------|------|------|
| 와일드카드(접두사) | `key = value*` | `host = prod*` |
| Regex(전체) | `key ~ "pattern"` | `host ~ "^prod-[0-9]+"` |

단순한 접두사 매칭에는 와일드카드를 사용하고, 더 복잡한 패턴이 필요하면 regex를 사용하세요.

## 예제

```
# Regex: "error" 또는 "fail"이 포함된 메시지
message ~ "error|fail"

# Regex: /api/v2로 시작하는 경로
path ~ "^/api/v2"

# 부정된 regex: health check 경로 제외
path !~ "^/health"

# 와일드카드: "prod"로 시작하는 host
host = prod*

# 조합 예제
host = prod* and message ~ "timeout.*" and path !~ "^/health"
```
