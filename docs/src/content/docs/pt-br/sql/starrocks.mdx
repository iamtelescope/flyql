## Tipos complexos aninhados

O gerador StarRocks trata um caminho segmentado como pertencente a um único tipo contêiner — atualmente não desce por aninhamentos mistos como `Map(K, Struct)`, `Array(Map)` ou campos `Struct` que sejam eles próprios `Map`/`Array`. Para essas colunas, achate o caminho ou exponha o valor interno como uma coluna separada.

### Configuração de Colunas

{/* TODO: native-speaker review */}
Colunas text/varchar cujo conteúdo é JSON válido são declaradas com o tipo bruto sintético `jsonstring` (igual ao ClickHouse):

```go
// Go
columns := map[string]*starrocks.Column{
    "message": starrocks.NewColumn(starrocks.ColumnDef{Name: "message", Type: "VARCHAR(255)"}),
    "data":    starrocks.NewColumn(starrocks.ColumnDef{Name: "data", Type: "JSON"}),      // JSON nativo
    "log":     starrocks.NewColumn(starrocks.ColumnDef{Name: "log", Type: "jsonstring"}),       // JSON string
}
```

```javascript
// JavaScript
const columns = {
    message: newColumn({ name: "message", type: "VARCHAR(255)" }),
    data:    newColumn({ name: "data", type: "JSON" }),      // JSON nativo
    log:     newColumn({ name: "log", type: "jsonstring" }),       // JSON string
}
```
