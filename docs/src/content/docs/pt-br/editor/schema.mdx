---
title: Configuração de Schema
description: Estrutura de schema de colunas para o editor FlyQL
---

A prop `columns` define o schema de dados que direciona as sugestões do editor. As chaves são nomes de colunas, os valores são definições de colunas.

## Schema Básico

```javascript
import { ColumnSchema } from 'flyql-vue'

const columns = ColumnSchema.fromPlainObject({
    level: { type: 'enum', suggest: true, autocomplete: true, values: ['debug', 'info', 'error'] },
    service: { type: 'string', suggest: true, autocomplete: true },
    status_code: { type: 'number', suggest: true },
    host: { type: 'string', suggest: true },
})
```

## Campos da Definição de Coluna

| Campo | Tipo | Padrão | Descrição |
|-------|------|---------|-------------|
| `type` | `string` | — | Tipo da coluna: `'string'`, `'number'`, `'enum'`, `'object'` |
| `suggest` | `boolean` | `true` | Mostrar esta coluna nas sugestões |
| `autocomplete` | `boolean` | `false` | Ativar autocomplete de valores para esta coluna |
| `values` | `string[]` ou `number[]` | — | Valores estáticos para autocomplete (usados antes do callback `onAutocomplete`) |
| `children` | `object` | — | Definições de colunas aninhadas para colunas do tipo objeto |
| `display_name` | `string` | `''` | Nome de exibição amigável opcional para a coluna. Em colunas parseadas, o alias sobrescreve este valor |

## Tipos de Colunas

### `string`

Coluna de texto geral. Todos os operadores de comparação estão disponíveis, incluindo regex (`~`, `!~`).

```javascript
{ type: 'string', suggest: true }
```

### `number`

Coluna numérica. Operadores regex (`~`, `!~`) são excluídos das sugestões de operadores.

```javascript
{ type: 'number', suggest: true }
```

### `enum`

Coluna enumerada com um conjunto fixo de valores. Assim como `number`, operadores regex são excluídos. Use `values` para fornecer o conjunto:

```javascript
{ type: 'enum', suggest: true, autocomplete: true, values: ['debug', 'info', 'warning', 'error', 'critical'] }
```

### `object`

Coluna aninhada com filhos. Quando o usuário digita o nome da coluna seguido de `.`, as colunas filhas são sugeridas:

```javascript
{
    type: 'object',
    suggest: true,
    children: {
        labels: {
            type: 'object',
            suggest: true,
            children: {
                tier: { type: 'string', suggest: true },
                env: { type: 'string', suggest: true },
            },
        },
        version: { type: 'string', suggest: true },
    },
}
```

## Autocomplete de Valores

### Valores Estáticos

Forneça um array `values` para colunas com opções conhecidas:

```javascript
{
    level: {
        type: 'enum',
        suggest: true,
        autocomplete: true,
        values: ['debug', 'info', 'warning', 'error', 'critical'],
    },
    status_code: {
        type: 'number',
        suggest: true,
        autocomplete: true,
        values: [200, 201, 204, 400, 404, 500],
    },
}
```

### Valores Assíncronos

Defina `autocomplete: true` sem `values` e forneça um callback `onAutocomplete` no componente do editor. O callback é chamado quando o usuário chega à posição de valor daquela coluna.

```javascript
// Definição da coluna — sem valores estáticos
{ type: 'string', suggest: true, autocomplete: true }

// Callback do editor busca valores da API
async function onAutocomplete(key, value) {
    const resp = await fetch(`/api/values?column=${key}`)
    const data = await resp.json()
    return { items: data.values }
}
```

## Colunas Aninhadas (JSONSchema)

Defina estruturas aninhadas usando `children` em colunas do tipo objeto:

```javascript
const columns = ColumnSchema.fromPlainObject({
    metadata: {
        type: 'object',
        suggest: true,
        children: {
            labels: {
                type: 'object',
                suggest: true,
                children: {
                    tier: { type: 'string', suggest: true, autocomplete: true, values: ['dev', 'staging', 'prod'] },
                    env: { type: 'string', suggest: true, autocomplete: true },
                },
            },
            version: { type: 'string', suggest: true },
        },
    },
})
```

Digitar `metadata.` sugere `labels` e `version`. Digitar `metadata.labels.` sugere `tier` e `env`. Não há limite de profundidade.

## Descoberta Remota de Chaves (Sem Schema)

Para colunas objeto cujas chaves aninhadas não são conhecidas no momento da definição do schema, omita `children` e forneça um callback `onKeyDiscovery`:

```javascript
const columns = ColumnSchema.fromPlainObject({
    request: { type: 'object', suggest: true }, // sem children — aciona a descoberta
}

async function onKeyDiscovery(columnName, segments) {
    const resp = await fetch(`/api/keys?column=${columnName}&path=${segments.join(',')}`)
    const data = await resp.json()
    return data.keys // Array<{ name: string, type?: string, hasChildren?: boolean }>
})
```

Quando o usuário digita `request.`, o callback é chamado com `('request', ['request'])`. Chaves descobertas são armazenadas em cache por sessão de ativação.

Se uma coluna tem tanto `children` quanto `onKeyDiscovery`, os `children` estáticos têm prioridade.

## Exemplo Completo

```javascript
const columns = ColumnSchema.fromPlainObject({
    // Enum com valores estáticos
    level: { type: 'enum', suggest: true, autocomplete: true, values: ['debug', 'info', 'error'] },

    // String com autocomplete assíncrono
    service: { type: 'string', suggest: true, autocomplete: true },

    // Número — exclui operadores regex
    status_code: { type: 'number', suggest: true, autocomplete: true, values: [200, 400, 500] },

    // String simples
    host: { type: 'string', suggest: true },

    // Aninhado com filhos conhecidos
    metadata: {
        type: 'object',
        suggest: true,
        children: {
            labels: {
                type: 'object',
                suggest: true,
                children: {
                    tier: { type: 'string', suggest: true, autocomplete: true, values: ['dev', 'staging', 'prod'] },
                    env: { type: 'string', suggest: true, autocomplete: true },
                },
            },
            version: { type: 'string', suggest: true },
        },
    },

    // Sem schema — chaves aninhadas descobertas em tempo de execução
    request: { type: 'object', suggest: true },
})
```
