---
title: Трансформеры
description: Преобразование значений колонок перед сравнением с помощью функций, объединённых через `|`
---

Трансформеры изменяют значения колонок перед сравнением в запросе или перед генерацией SQL. Они используют синтаксис `|` и применяются слева направо.

## Синтаксис `|`

```
message|upper = "ERROR"
```

## Цепочки

```
message|lower|len > 100
```

Сначала выполняется `lower`, затем `len`.

## Система типов

Каждый трансформер объявляет входной и выходной тип. Цепочки должны быть типово совместимыми.

| Тип | Описание |
|-----|----------|
| `string` | текст |
| `int` | целые числа |
| `float` | числа с плавающей точкой |
| `array` | массив значений |

## Встроенные трансформеры

| Трансформер | Вход | Выход | Описание |
|-------------|------|-------|----------|
| `upper` | string | string | перевод в верхний регистр |
| `lower` | string | string | перевод в нижний регистр |
| `len` | string | int | длина строки |
| `split` | string | array | разделение по разделителю |

## Примеры

```
status|upper = "ERROR"
message|lower|len > 100
host|lower != "localhost"
path|len > 0
tags|split(",") has "urgent"
```
