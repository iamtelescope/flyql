FlyQL は 3 つの言語で利用できます。利用するスタックに合ったものを選んでください。

| 言語 | インストール | Parser | Generator | Matcher |
|------|--------------|--------|-----------|---------|
| [Go](/ja/getting-started/go/) | `go get` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [Python](/ja/getting-started/python/) | `pip install` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |
| [JavaScript](/ja/getting-started/javascript/) | `npm install` | Yes | ClickHouse, PostgreSQL, StarRocks | Yes |

各ガイドでは、インストール、クエリの parse、SQL 生成、そして利用可能な場合はメモリ上でのマッチ処理までを説明します。
