---
title: 快速开始
description: 安装 FlyQL，并在 Go、Python 或 JavaScript 中开始过滤数据
---

FlyQL 提供三种语言实现。请选择与你当前技术栈最匹配的版本：

| 语言 | 安装方式 | 解析器 | 生成器 | 匹配器 |
|------|----------|--------|--------|--------|
| [Go](/zh/getting-started/go/) | `go get` | 支持 | ClickHouse、PostgreSQL、StarRocks | 支持 |
| [Python](/zh/getting-started/python/) | `pip install` | 支持 | ClickHouse、PostgreSQL、StarRocks | 支持 |
| [JavaScript](/zh/getting-started/javascript/) | `npm install` | 支持 | ClickHouse、PostgreSQL、StarRocks | 支持 |

每个语言指南都会介绍安装方式、查询解析、SQL 生成，以及在可用情况下的内存数据匹配。
