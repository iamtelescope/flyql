title: Primeiros Passos
description: Instalar o FlyQL e começar a filtrar dados em Go, Python ou JavaScript
---

O FlyQL está disponível em três linguagens. Escolha a que corresponde à sua stack:

| Linguagem | Instalação | Parser | Geradores | Matcher |
|----------|---------|--------|-----------|---------|
| [Go](/pt/getting-started/go/) | `go get` | Sim | ClickHouse, PostgreSQL, StarRocks | Sim |
| [Python](/pt/getting-started/python/) | `pip install` (git) | Sim | ClickHouse, PostgreSQL, StarRocks | Sim |
| [JavaScript](/pt/getting-started/javascript/) | `npm install` | Sim | ClickHouse, PostgreSQL, StarRocks | Sim |

Cada guia abrange instalação, análise de uma consulta, geração de SQL e (quando disponível) correspondência de dados em memória.
