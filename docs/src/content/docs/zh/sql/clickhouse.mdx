内容为 JSON 的 text/varchar 列用合成原始类型 `jsonstring` 声明：

```go
columns := map[string]*clickhouse.Column{
    "message": clickhouse.NewColumn(clickhouse.ColumnDef{Name: "message", Type: "String"}),
    "data":    clickhouse.NewColumn(clickhouse.ColumnDef{Name: "data", Type: "JSON"}),
    "log":     clickhouse.NewColumn(clickhouse.ColumnDef{Name: "log", Type: "jsonstring"}),
}
```

```javascript
const columns = {
    message: newColumn({ name: "message", type: "String" }),
    data:    newColumn({ name: "data", type: "JSON" }),
    log:     newColumn({ name: "log", type: "jsonstring" }),
}
```
